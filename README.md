# Breakout

Atari's classic Breakout game with levels, speed modes, lives and score count.

![Alt img](https://i.imgur.com/vhtRGoQ.png)

## Installation

1. `git clone https://github.com/madzadev/breakout.git`

2. `cd breakout`

3. `npm install`

4. `npm run dev`
